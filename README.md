# Finance Dashboard

A clean and interactive **Finance Dashboard** built with **React, TypeScript, and Tailwind CSS** to track income, expenses, and financial insights.

---

## Features

* **Dashboard Overview** – Balance, income, and expense summaries with charts
* **Transactions** – Add, view, filter, search, and export transactions
* **Role-Based Access** – Viewer (read-only) & Admin (full control)
* **Insights** – Spending trends, savings rate, and analytics
* **Dark Mode** – Toggle between light and dark themes
* **Local Storage** – Data persists across sessions
* **Responsive UI** – Works on mobile, tablet, and desktop

---

## Tech Stack

* React + TypeScript
* Tailwind CSS
* Recharts
* Context API
* Vite

---

## Setup

```bash
npm install
npm run dev
```

---

## Roles

* **Viewer** → View-only access
* **Admin** → Add & delete transactions

---

## Key Functionalities

* Add & manage transactions
* Filter & search data
* Sort by date/amount
* Export transactions as CSV
